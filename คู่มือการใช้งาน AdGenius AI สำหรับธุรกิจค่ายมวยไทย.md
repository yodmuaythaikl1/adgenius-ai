## บทนำ

AdGenius AI เป็นแพลตฟอร์ม AI สำหรับการยิงโฆษณาแบบครบวงจรที่ออกแบบมาเพื่อช่วยให้ธุรกิจค่ายมวยไทยแบบออกกำลังกายสามารถเข้าถึงกลุ่มเป้าหมายได้อย่างแม่นยำและมีประสิทธิภาพ ด้วยความสามารถในการวิเคราะห์กลุ่มเป้าหมาย การสร้างสรรค์โฆษณา และการเพิ่มประสิทธิภาพแคมเปญ AdGenius AI จะช่วยให้คุณประหยัดงบประมาณและได้ผลลัพธ์ที่ดีที่สุด

คู่มือนี้จะแนะนำวิธีการใช้งาน AdGenius AI สำหรับธุรกิจค่ายมวยไทยแบบออกกำลังกาย ตั้งแต่การติดตั้งระบบไปจนถึงการวิเคราะห์ผลและปรับปรุงแคมเปญ

## การติดตั้งระบบ

### ความต้องการของระบบ
- Python 3.8 หรือสูงกว่า
- Node.js 14 หรือสูงกว่า
- MongoDB 4.4 หรือสูงกว่า
- ระบบปฏิบัติการ: Windows, macOS, หรือ Linux

### ขั้นตอนการติดตั้ง

1. **ดาวน์โหลดและติดตั้ง AdGenius AI**

```bash
# โคลนโปรเจค
git clone https://github.com/adgenius-ai/adgenius-platform.git

# เข้าไปยังไดเรกทอรีของโปรเจค
cd adgenius-platform

# ติดตั้งแพ็คเกจที่จำเป็นสำหรับ Backend
pip install -r requirements.txt

# ติดตั้งแพ็คเกจที่จำเป็นสำหรับ Frontend
cd frontend
npm install
cd ..
```

2. **ตั้งค่าไฟล์ .env**

สร้างไฟล์ .env ในไดเรกทอรีหลักของโปรเจคและกำหนดค่าต่อไปนี้:

```
# การตั้งค่าฐานข้อมูล
MONGODB_URI=mongodb://localhost:27017/adgenius

# การตั้งค่า API Key
FACEBOOK_API_KEY=your_facebook_api_key
INSTAGRAM_API_KEY=your_instagram_api_key
TIKTOK_API_KEY=your_tiktok_api_key
SHOPEE_API_KEY=your_shopee_api_key
OPENAI_API_KEY=your_openai_api_key

# การตั้งค่าอื่นๆ
JWT_SECRET=your_jwt_secret
PORT=5000
```

3. **เริ่มต้นใช้งานระบบ**

```bash
# เริ่มต้น Backend
python run.py

# เริ่มต้น Frontend (ในอีกเทอร์มินัลหนึ่ง)
cd frontend
npm start
```

หลังจากเริ่มต้นระบบแล้ว คุณสามารถเข้าถึง AdGenius AI ได้ที่ http://localhost:3000

## การเริ่มต้นใช้งาน

### การลงทะเบียนและเข้าสู่ระบบ

1. เปิดเบราว์เซอร์และไปที่ http://localhost:3000
2. คลิกที่ "ลงทะเบียน" และกรอกข้อมูลที่จำเป็น
3. หลังจากลงทะเบียนแล้ว เข้าสู่ระบบด้วยอีเมลและรหัสผ่านของคุณ

### การตั้งค่าบัญชีค่ายมวยไทย

1. หลังจากเข้าสู่ระบบแล้ว ไปที่ "การตั้งค่า" > "ข้อมูลธุรกิจ"
2. กรอกข้อมูลเกี่ยวกับค่ายมวยไทยของคุณ:
   - ชื่อค่ายมวย
   - คำอธิบาย
   - คุณสมบัติ (เช่น คลาสมวยไทยเพื่อการออกกำลังกาย, โปรแกรมลดน้ำหนักด้วยมวยไทย)
   - ราคา
   - ตำแหน่งที่ตั้ง
3. อัปโหลดโลโก้และรูปภาพของค่ายมวย
4. คลิก "บันทึก" เพื่อบันทึกข้อมูล

### การเชื่อมต่อกับแพลตฟอร์มโฆษณา

1. ไปที่ "การตั้งค่า" > "การเชื่อมต่อ"
2. เชื่อมต่อกับแพลตฟอร์มโฆษณาที่คุณต้องการใช้:
   - Facebook
   - Instagram
   - TikTok
   - Shopee
3. ทำตามขั้นตอนการเชื่อมต่อสำหรับแต่ละแพลตฟอร์ม

## การกำหนดกลุ่มเป้าหมาย

AdGenius AI มีโมดูล AI สำหรับการกำหนดกลุ่มเป้าหมายที่ช่วยให้คุณสามารถระบุกลุ่มเป้าหมายที่เหมาะสมสำหรับค่ายมวยไทยของคุณได้

### การวิเคราะห์กลุ่มเป้าหมาย

1. ไปที่ "แคมเปญ" > "สร้างแคมเปญใหม่"
2. เลือก "วิเคราะห์กลุ่มเป้าหมาย"
3. ระบบจะวิเคราะห์ข้อมูลค่ายมวยไทยของคุณและแนะนำกลุ่มเป้าหมายที่เหมาะสม:
   - **คนท้องถิ่นที่ต้องการออกกำลังกาย**: กลุ่มคนในพื้นที่ที่สนใจการออกกำลังกายและสุขภาพ
   - **นักท่องเที่ยวและชาวต่างชาติ**: กลุ่มนักท่องเที่ยวที่สนใจประสบการณ์มวยไทยแบบดั้งเดิม
   - **ผู้ที่ต้องการลดน้ำหนัก**: กลุ่มคนที่ต้องการลดน้ำหนักและมีสุขภาพที่ดีขึ้น

### การปรับแต่งกลุ่มเป้าหมาย

1. เลือกกลุ่มเป้าหมายที่คุณต้องการ
2. ปรับแต่งพารามิเตอร์ต่างๆ:
   - ช่วงอายุ
   - เพศ
   - ความสนใจ
   - พฤติกรรม
   - ตำแหน่งที่ตั้ง
3. คลิก "บันทึก" เพื่อบันทึกการตั้งค่ากลุ่มเป้าหมาย

### การค้นหาคำสำคัญ

1. ไปที่ "เครื่องมือ" > "ค้นหาคำสำคัญ"
2. เลือกกลุ่มเป้าหมายที่คุณต้องการค้นหาคำสำคัญ
3. ระบบจะแนะนำคำสำคัญที่เกี่ยวข้องกับค่ายมวยไทยและกลุ่มเป้าหมายที่คุณเลือก
4. เลือกคำสำคัญที่คุณต้องการใช้และคลิก "เพิ่มไปยังแคมเปญ"

## การสร้างสรรค์โฆษณา

AdGenius AI มีโมดูล AI สำหรับการสร้างสรรค์โฆษณาที่ช่วยให้คุณสามารถสร้างโฆษณาที่น่าสนใจและมีประสิทธิภาพสำหรับค่ายมวยไทยของคุณได้

### การสร้างพาดหัวโฆษณา

1. ไปที่ "แคมเปญ" > "สร้างโฆษณา"
2. เลือก "สร้างพาดหัว"
3. เลือกกลุ่มเป้าหมายที่คุณต้องการสร้างพาดหัว
4. ระบบจะสร้างพาดหัวโฆษณาที่เหมาะสมสำหรับค่ายมวยไทยและกลุ่มเป้าหมายที่คุณเลือก
5. เลือกพาดหัวที่คุณต้องการใช้หรือปรับแต่งตามต้องการ

### การสร้างคำอธิบายโฆษณา

1. เลือก "สร้างคำอธิบาย"
2. เลือกกลุ่มเป้าหมายที่คุณต้องการสร้างคำอธิบาย
3. ระบบจะสร้างคำอธิบายโฆษณาที่เหมาะสมสำหรับค่ายมวยไทยและกลุ่มเป้าหมายที่คุณเลือก
4. เลือกคำอธิบายที่คุณต้องการใช้หรือปรับแต่งตามต้องการ

### การสร้างภาพโฆษณา

1. เลือก "สร้างภาพ"
2. เลือกกลุ่มเป้าหมายที่คุณต้องการสร้างภาพ
3. ระบบจะแนะนำคำแนะนำสำหรับการสร้างภาพโฆษณาที่เหมาะสมสำหรับค่ายมวยไทยและกลุ่มเป้าหมายที่คุณเลือก
4. คลิก "สร้างภาพ" เพื่อสร้างภาพโฆษณาตามคำแนะนำ
5. ปรับแต่งภาพตามต้องการและคลิก "บันทึก"

## การยิงโฆษณาบนแพลตฟอร์มต่างๆ

AdGenius AI รองรับการยิงโฆษณาบนแพลตฟอร์มต่างๆ ได้แก่ Facebook, Instagram, TikTok และ Shopee

### การยิงโฆษณาบน Facebook และ Instagram

1. ไปที่ "แคมเปญ" > "สร้างแคมเปญใหม่"
2. เลือก "Facebook และ Instagram"
3. เลือกกลุ่มเป้าหมายและโฆษณาที่คุณสร้างไว้
4. กำหนดงบประมาณและระยะเวลาของแคมเปญ
5. เลือกช่วงเวลาที่เหมาะสมสำหรับการแสดงโฆษณา (ระบบจะแนะนำช่วงเวลาที่เหมาะสมสำหรับกลุ่มเป้าหมายของคุณ)
6. คลิก "เริ่มแคมเปญ" เพื่อเริ่มการยิงโฆษณา

### การยิงโฆษณาบน TikTok

1. ไปที่ "แคมเปญ" > "สร้างแคมเปญใหม่"
2. เลือก "TikTok"
3. เลือกกลุ่มเป้าหมายและโฆษณาที่คุณสร้างไว้
4. กำหนดงบประมาณและระยะเวลาของแคมเปญ
5. เลือกช่วงเวลาที่เหมาะสมสำหรับการแสดงโฆษณา
6. คลิก "เริ่มแคมเปญ" เพื่อเริ่มการยิงโฆษณา

### การยิงโฆษณาบน Shopee

1. ไปที่ "แคมเปญ" > "สร้างแคมเปญใหม่"
2. เลือก "Shopee"
3. เลือกสินค้าหรือบริการที่คุณต้องการโฆษณา
4. เลือกกลุ่มเป้าหมายและโฆษณาที่คุณสร้างไว้
5. กำหนดงบประมาณและระยะเวลาของแคมเปญ
6. คลิก "เริ่มแคมเปญ" เพื่อเริ่มการยิงโฆษณา

## การวิเคราะห์ผลและปรับปรุงแคมเปญ

AdGenius AI มีเครื่องมือสำหรับการวิเคราะห์ผลและปรับปรุงแคมเปญที่ช่วยให้คุณสามารถติดตามประสิทธิภาพของแคมเปญและปรับปรุงให้ดียิ่งขึ้นได้

### การติดตามประสิทธิภาพแคมเปญ

1. ไปที่ "แคมเปญ" > "ประสิทธิภาพ"
2. เลือกแคมเปญที่คุณต้องการติดตาม
3. ดูข้อมูลประสิทธิภาพของแคมเปญ:
   - จำนวนการแสดงผล
   - จำนวนการคลิก
   - อัตราการคลิก (CTR)
   - ต้นทุนต่อการคลิก (CPC)
   - การแปลง (Conversions)
   - ต้นทุนต่อการแปลง (CPA)
   - ROI

### การวิเคราะห์กลุ่มเป้าหมาย

1. ไปที่ "แคมเปญ" > "วิเคราะห์กลุ่มเป้าหมาย"
2. เลือกแคมเปญที่คุณต้องการวิเคราะห์
3. ดูข้อมูลเชิงลึกเกี่ยวกับกลุ่มเป้าหมายของคุณ:
   - ข้อมูลประชากร (อายุ, เพศ)
   - ความสนใจ
   - พฤติกรรม
   - ตำแหน่งที่ตั้ง
4. ใช้ข้อมูลนี้เพื่อปรับปรุงการกำหนดกลุ่มเป้าหมายของคุณ

### การปรับปรุงแคมเปญ

1. ไปที่ "แคมเปญ" > "ปรับปรุง"
2. เลือกแคมเปญที่คุณต้องการปรับปรุง
3. ระบบจะวิเคราะห์ประสิทธิภาพของแคมเปญและให้คำแนะนำในการปรับปรุง:
   - การปรับปรุงกลุ่มเป้าหมาย
   - การปรับปรุงโฆษณา
   - การปรับปรุงงบประมาณ
   - การปรับปรุงตารางเวลา
4. เลือกคำแนะนำที่คุณต้องการใช้และคลิก "ปรับปรุงแคมเปญ"

## คำถามที่พบบ่อย

### AdGenius AI เหมาะสำหรับค่ายมวยไทยขนาดเล็กหรือไม่?
ใช่ AdGenius AI เหมาะสำหรับค่ายมวยไทยทุกขนาด ไม่ว่าจะเป็นค่ายมวยขนาดเล็กหรือขนาดใหญ่ ระบบสามารถปรับให้เหมาะสมกับงบประมาณและความต้องการของคุณได้

### ฉันจำเป็นต้องมีความรู้ด้านการตลาดดิจิทัลหรือไม่?
ไม่จำเป็น AdGenius AI ออกแบบมาให้ใช้งานง่ายสำหรับผู้ใช้ทุกระดับ ไม่ว่าคุณจะมีประสบการณ์ด้านการตลาดดิจิทัลหรือไม่ก็ตาม

### AdGenius AI สามารถช่วยลดต้นทุนการโฆษณาได้อย่างไร?
AdGenius AI ช่วยลดต้นทุนการโฆษณาด้วยการกำหนดกลุ่มเป้าหมายที่แม่นยำ การสร้างโฆษณาที่มีประสิทธิภาพ และการเพิ่มประสิทธิภาพแคมเปญอย่างต่อเนื่อง ซึ่งช่วยให้คุณได้ผลลัพธ์ที่ดีที่สุดจากงบประมาณที่มี

### ฉันสามารถใช้ AdGenius AI กับธุรกิจอื่นนอกเหนือจากค่ายมวยไทยได้หรือไม่?
ใช่ AdGenius AI สามารถปรับให้เหมาะสมกับธุรกิจประเภทต่างๆ ได้ ไม่เพียงแต่ค่ายมวยไทยเท่านั้น

### AdGenius AI รองรับภาษาอื่นนอกจากภาษาไทยหรือไม่?
ใช่ AdGenius AI รองรับหลายภาษา รวมถึงภาษาอังกฤษ ภาษาจีน และภาษาอื่นๆ
